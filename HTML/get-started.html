Get Started

Web User

Just click once on the Bookmark icon on the web page and get it saved in your account. After saving the bookmark user will see a small green tick mark on the icon to confirm that bookmark has been saved. Bookmark icon will be disabled for the perticular page after bookmarking and also if the user is revisiting the same page.

If the user still clicks on the disabled icon then it eill open a small window to edit the Bookmark saved.By default all bookmarks will be saved under default folder until the user edits it.

Site Owner

Register your website with your site URL and name. Then generate the script code by selecting the desired icon size needed to match your application and by selecting if you want to display the total bookmark count.

Copy and paste the code in your application when you want the bookmark icon to be displayed on your web pages. It is recommended that you place this icon on every unique webpage of your application in such a way that the icon is highlighted.
